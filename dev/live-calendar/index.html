live-calendar

This is a event calendar plugin for wordpress, it works similar to microsoft live calendar .You can make a calendar page or calendar sidebar or a upcoming event list.

Latest Version : V1.9.6, 14 Dec, 2012 (Fixed warning issue after upgrade to WP3.5)

Reminder: Back up your db before upgrade!
