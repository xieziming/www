为博客添加flash批量上传功能

11 July 2010 Categoried in 杂记 0 Preview : 243 , Read : 1153 Index : 4187

存放博客图片主机的FTP上传速度让我郁闷的不是一天两天了，HTTP上传与FTP上传咋就差距这么大呢！很喜欢Dokuwiki带的flash批量上传插件，那速度真是省心啊！

昨天搞了一天终于把它移植了过来，顺便加了自定义上传目录的功能，这下子图片上传方便快捷多了。

模块主要有两部分组成，一个是前端的flash（值得称道的是它是外带AS的，以后修改就方便多了），后台用PHP来处理，其实很简单，要做的事就是把上传的临时文件MOVE到正确的路径。

<?php
$upload_dir= $_SERVER['DOCUMENT_ROOT']."/upload/".$_GET['key'];
mkdirs($upload_dir);
 
function mkdirs($dir) {  
	if(!is_dir($dir)) {  
		if(!mkdirs(dirname($dir))){   
				return false;  
		}   
		if(!mkdir($dir,0777)){  
			return false;   
		}  
	}   
	return true; 
}   
 
$attachment = $_FILES['Filedata'];
$filename = $attachment['name'];
$fileext = substr(strrchr($filename,'.'),1);
 
$path = $upload_dir."/".$filename;
move_uploaded_file($attachment['tmp_name'],$path);
 
?>
