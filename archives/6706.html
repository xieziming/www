3. Display post views
Add the following code to you template’s single.php and index.php or another place where you want to show it.

get_post_views($view_type,$time_span,$output_type,$format,$post_id,$by_cache)

Available parameter:
$view_type: ‘normal’, ‘robot’
$time_span: ‘today’, ‘week’, ‘month’, ‘halfyear’, ‘year’, ‘total’. ‘total’ as default.
$output_type: ‘content’,’excerpt’. ‘content’ as default.
$format: true, false. true as default, with ‘,’ as the separator.
$post_id: any available post id. ‘ ‘ as default,given automatically.
$by_cache: needless, just for inner invoking.

4. Display post views widget

You can add a ‘post-views’ widget to you sidebar, it includes the following widgets:

‘most view’, most view by categories’, ‘most view by tags’
‘least view’ , ‘least view by categories’, ‘least view by tags’
‘latest view’ , ‘latest view by categories’, ‘latest view by tags’
‘oldest view’ , ‘oldest view by categories’, ‘oldest view by tags’

5. External Invoking Functions

show_post_views_list

function show_post_views_list($mode = '', $view_type = 'normal', $output_type = 'content', $time_span = 'total',
$order_type = 'views', $order_by = 'DESC', $limit = 10, $chars = 0, $category_id = 0, $tag_id = 0, $before = '<li>', 
$after = '</li>'){
}

Available parameter :
$mode: filtering the post type. available: ‘both’, ‘page’, ‘post’.

$view_type: filtering the vew type. available: ‘normal’, ‘robot’.

$output_type: filter the output type. available: ‘content’,’robot’.

$time_span: filter the visited time. available: ‘today’, ‘week’, ‘month’, ‘halfyear’, ‘year’, ‘total’.

$order_type: resultset order by. available: ‘views’, ‘latest view time’.

$order_by: resultset order by. available: ‘DESC’, ‘ASC’.

$limit: how many records to return? available: all available int type value.

$chars: the title width, if the title is too long, it can cut it replace with ‘…’, type 0 if don’t want it to be cut.

$category_id: filtering the post’s category, can be a array or a int value. e.g. array(8,17,21), 8, 17.
type it as 0, if you don’t want to filter category.

$tag_id: filtering the post’s tag,can be a array or a int value. e.g. array(8,17,21), 8, 17.
type it as 0, if you don’t want to filter category.

$before: what to show before ‘post title’, maybe you like to add ‘< li >‘ or ‘< p >‘ before outputting the content.
$after: what to show after ‘post title’, maybe you like to add ‘< li >‘ or ‘< p >‘ after outputting the content.

get_post_views_list

Functionality : return a post-views array including ‘title’, ‘views’, ‘time’.

function get_post_views_list($mode = '', $view_type = 'normal', $output_type = 'content', $time_span = 'total',
$order_type = 'views', $order_by = 'DESC', $limit = 10, $chars = 0, $category_id = 0, $tag_id = 0)

Available parameter : like above except ‘$before’ and ‘after’.

query_posts

< ?php
$args = array(
'showposts' => 10,
'v_sortby'  => 'views',
'v_orderby' => 'DESC',
'v_timespan' => 'week',
'v_outtype' => 'content',
'category__in' => array(78,9,203)
 );	
query_posts($args);
echo '<p>BY Category & Year</p>';
while (have_posts()){
     the_post(); 
     echo '<li><a href="'.get_permalink().'" target="_blank" >'.$post->post_title.' </a>(views : '.$post->views.')</li>';
}
?>

Available parameter :
The following are post-views’ query keywords, for more , pls visit 【query_posts on wordpress】

v_sortby : ‘views’, ‘robot’.
v_ordertype : (optional) ‘views’, default. ‘latest_view_time’, if you want to order by visited time.
v_orderby : ‘DESC’, ‘ASC’.
v_outtype: ‘content’,’excerpt.’ ‘content’ as default.
v_timespan : (optional) ‘today’, ‘week’, ‘month’, ‘halfyear’, ‘year’, ‘total’.
